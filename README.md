# הדרכה מחוז ירושלים

אתר פרימיום בעברית מלאה עם עיצוב יוקרתי, RTL מושלם, ופונט Heebo.

## 🚀 הרצה מהירה

### התקנה ראשונית
```bash
npm install
```

### הרצת שרת פיתוח (לייב)
```bash
npm run dev
```
האתר יהיה זמין ב: **http://localhost:3000**

### מעקב אוטומטי ועדכון ל-GitHub
```bash
npm run watch
```
סקריפט זה עוקב אחר כל השינויים ומעדכן אוטומטית את ה-GitHub תוך 5 שניות.

### הרצת הכל יחד
```bash
npm start
```
מריץ את שרת הפיתוח ואת המעקב האוטומטי יחד.

## 📡 סנכרון אוטומטי עם GitHub

המערכת מוגדרת לעדכן אוטומטית את ה-GitHub repository (`yanivmizrachiy/hadracha`) בכל שינוי:

- ✅ כל שינוי בקובץ → commit אוטומטי
- ✅ Push אוטומטי ל-GitHub
- ✅ עובד ברקע כל הזמן

## 🛠️ פיתוח

- **Vite** - שרת פיתוח מהיר עם Hot Module Replacement
- **Auto-sync** - עדכון אוטומטי ל-GitHub
- **Watch Mode** - מעקב אחר שינויים בזמן אמת

## 📝 הערות

- השרת רץ על פורט 3000
- כל שינוי נשמר אוטומטית ל-GitHub
- אין צורך ב-commit ידני - הכל אוטומטי!
